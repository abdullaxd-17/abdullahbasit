# Abdullah Basit Portfolio Website

This repository contains a one-page personal portfolio website for Abdullah Basit.

## Files

- `index.html` — main portfolio website
- `.nojekyll` — keeps GitHub Pages simple and prevents Jekyll processing

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html` and `.nojekyll` to the root of the repository.
3. Commit the files.
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, select:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
6. Save.
7. Your website will be available from the GitHub Pages link shown there.
